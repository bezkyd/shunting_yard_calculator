# 🧮 Calculator App
A modern calculator built with JavaScript, featuring advanced expression parsing and a clean UI.

---
## 🚀 Features
* Basic arithmetic operations (+, −, ×, ÷)
* Operator precedence handling
* Expression parsing using the **Shunting Yard algorithm**
* Support for **implicit multiplication**
* Floating-point calculation fixes
* Input validation (e.g. prevents multiple decimal points)
* Keyboard input support
* Clickable display
* Calculation history
* Light / Dark theme toggle
* Responsive UI 
---

## 🧠 Core Logic
The calculator uses the **Shunting Yard algorithm** to:
* Convert infix expressions → postfix (RPN)
* Correctly handle operator precedence
* Evaluate complex mathematical expressions
